# スマホのドロップダウンメニュー

```javascript
        <script>
        $(function() {
            $(".on").click(function(){
                $(this).css("display","none"); //横線消える
                $(".off").css("display","block");　//×を表示
                $(".nav").slideToggle();　// ナビの表示
            })
            $(".off").click(function(){
                $(this).css("display","none");
                $(".on").css("display","block");
                $(".nav").slideToggle();
            })
        });
        </script>
```
