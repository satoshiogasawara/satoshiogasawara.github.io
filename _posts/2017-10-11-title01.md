# cssのハック

参考サイト   
http://sublimelife.hatenablog.com/entry/2017/02/24/140501   

Windows10のデフォルトブラウザ「Edge」のみに適用させるcssハックが必要になった   

```css
@supports (-ms-accelerator:true) {
	.css_selector {
	    background-color: red;
	}
}
```
↑今までのはもう効かない   

```css
@supports (-ms-ime-align:auto) {
	.css_selector {
	    background-color: red;
	}
}
```
