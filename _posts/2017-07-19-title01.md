# wordpress備忘録①

## カスタムメニューを有効にする方法の1つ

functions.phpに以下のように記述   
```php
<?php
add_theme_support('menus');
```

phpファイルに以下を記述   
```php
<?php wp_nav_menu(); ?>
```

すると   
外観 > メニュー   
で設定したものが表示される   

パラメーターの記入例   
```php
<?php wp_nav_menu(array(
    'menu' => 'mymenu',
    'container'=> 'div'
));
?>
```
