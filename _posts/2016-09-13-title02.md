# 滑らかスクロールでトップへ

```javascript
<script>
        $(document).ready(function() {
            var pagetop = $('.totop');
            pagetop.click(function () {
                $('body, html').animate({ scrollTop: 0 }, 500);
                return false;
            });
        });
</script>
```
```html
<p class="totop"><a href="#wrap" class="arrow"><img src="images/btn_pagetop.png" alt=""></a></p>
```
