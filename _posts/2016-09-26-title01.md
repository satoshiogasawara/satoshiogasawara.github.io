# スクロールしたらクラスを追加する   

スクロールしたらクラスを追加する   

```javascript
$(function($) {
	$(function () {
		$(window).scroll(function () {
			if ($(this).scrollTop() > 85) {
			$(".global_navi").addClass('fixmenu');
		} else {
			$(".global_navi").removeClass('fixmenu');
		}
		})
	})
});
```
